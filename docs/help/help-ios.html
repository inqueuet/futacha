はじめに

futacha は、ふたば☆ちゃんねるを快適に閲覧・書き込みするためのアプリです。

アプリの基本的な流れは次のとおりです。

  1. 見たい板を登録する
  2. 板を開くとスレッド一覧（カタログ）が表示される
  3. 気になるスレッドをタップして投稿を読む
  4. 必要に応じて書き込みや保存をする

板の管理

アプリを起動すると最初に表示される画面です。よく見る板を登録しておくと、すぐにアクセスできます。

画面の構成

  • 画面上部の左 - 三本線のアイコン（☰）をタップすると、閲覧履歴が左からスライドして表示されます
  • 画面上部の中央 - 「板一覧」などの画面タイトル
  • 画面上部の右 - 縦3点アイコン（⋮）をタップするとメニューが開きます
  • 画面の中央 - 登録した板がカード形式で並んでいます。タップすると板を開きます

板を追加する

操作：画面右上の ⋮ をタップ →「新規追加」を選ぶ → 板の名前とアドレスを入力 →「追加」をタップ
板のアドレスは https://○○.2chan.net/○○/futaba.php という形式です。Safari で板を開いたときのアドレスをコピーして貼り付けると簡単です。

板の順番を変える

操作：画面右上の ⋮ をタップ →「並び替え」を選ぶ → 各板のカードに表示される上下矢印ボタンで順番を変える

板を削除する

操作：画面右上の ⋮ をタップ →「削除」を選ぶ → 削除したい板のカードをタップ → 確認ダイアログで「削除」をタップ

その他のメニュー

  • 保存済み - 保存したスレッドの一覧を開く（ネットに繋がなくても読めます）
  • 設定 - アプリ全体の設定画面を開く

スレッド一覧（カタログ）

板をタップすると、その板にあるスレッドが画像つきの一覧で表示されます。

画面の構成

  • 画面上部の左 - 三本線のアイコン（☰）をタップすると閲覧履歴が開きます
  • 画面上部の中央 - 板の名前と、現在の並び順（「多い順」など）
  • 画面上部の右 - 虫眼鏡アイコン（検索）と ⋮（メニュー）
  • 画面の中央 - スレッドがタイル状またはリスト形式で並んでいます
  • 画面の下部 - 操作ボタンが横一列に並んでいます

スレッドを開く

操作：見たいスレッドのサムネイル画像やタイトルをタップ

スレッドを探す（検索）

操作：画面右上の虫眼鏡アイコンをタップ → 上部にキーワード入力欄が表示される → キーワードを入力すると一覧がリアルタイムで絞り込まれる → もう一度 ✕ をタップすると検索を終了

一覧を更新する

操作：画面下の「更新」ボタンをタップ。または、一覧を下に引っ張って離すと更新されます

並び順を変える

操作：画面下の「モード」ボタンをタップ → 表示されるリストから並び順を選ぶ
モードどんな順番？
カタログふたば標準の表示順
新順新しく立ったスレッドから
古順古いスレッドから
多い順書き込みが多いスレッドから
少ない順書き込みが少ないスレッドから
勢い順盛り上がっているスレッドから
そ順ふたば独自の並び順
監視登録した監視ワードに一致するスレッドだけ表示

表示をタイルとリストで切り替える

操作：画面右上の ⋮ をタップ →「表示の切り替え」を選ぶ

タイル表示の列数は 2〜8 で調整できます（初期設定は 5列）。

画面下のボタン一覧

以下のボタンが横一列に並んでいます（設定で表示・非表示や順番を変更できます）。

  • スレ立て - 新しいスレッドを作る
  • 先頭へ - 一覧の一番上に戻る
  • 更新 - 一覧を最新の状態に更新する
  • モード - 並び順を変える
  • 設定 - カタログの設定メニューを開く

設定メニュー（画面右上 ⋮ から開く）

  • 監視ワード - 気になるキーワードを登録しておくと、「監視」モードでそのキーワードを含むスレッドだけを表示
  • NG管理 - 特定のワードを含むスレッドをカタログから非表示に
  • 外部アプリ - 今見ている板を Safari で開く
  • 表示の切り替え - タイル⇔リスト、列数の調整
  • 先頭へ - 一覧の一番上に戻る
  • プライバシー - 画像にぼかしをかける（電車の中などで便利）

スレッドを読む

一覧からスレッドをタップすると、そのスレッドの投稿を読めます。

画面の構成

  • 画面上部の左 - ← ボタンでスレッド一覧に戻ります
  • 画面上部の中央 - スレッドのタイトル、板名、レス数
  • 画面上部の右 - 虫眼鏡アイコン（スレッド内検索）と ⋮（メニュー）
  • 画面の中央 - 投稿が上から順に表示されます。上下にスクロールして読みます
  • 画面の下部 - 操作ボタンが横一列に並んでいます

画面下のボタン一覧

以下のボタンが横一列に並んでいます（設定で変更できます）。

ボタンできること
返信書き込み画面を開く
先頭へ最初の投稿に戻る
末尾へ最新の投稿に移動する
更新新しい書き込みがないか確認する
ギャラリー画像・動画の一覧を開く
保存スレッドを端末に保存する
フィルター投稿をしぼり込む
設定追加メニューを開く

「設定」ボタンから開くメニュー

画面下の「設定」ボタンをタップすると、画面の下からメニューがスライドして表示されます。

  • NG管理 - 見たくない投稿を非表示にする設定（詳しくはこちら）
  • 外部アプリ - このスレッドを Safari で開く
  • 読み上げ - 投稿を音声で読み上げる
  • プライバシー - 画像にぼかしをかける

投稿をタップしたとき

操作：投稿の本文をタップ → その投稿を引用した返信画面が開きます

投稿を長押ししたとき

操作：投稿の本文を長押し → 画面の下からアクションメニューがスライドして表示されます
  • そうだね - その投稿に「そうだね」を送る
  • 削除 - 自分の投稿を削除する（パスワードが必要）
  • NG登録 - その投稿者を非表示リストに追加する
  • 引用選択 - 文章の一部を選んで引用する
  • リンクコピー - その投稿のアドレスをコピーする

画像をタップしたとき

操作：投稿の画像をタップ → 画像が拡大表示されます。2本指で拡大・縮小、1本指でスクロールできます。画面外をタップすると閉じます

投稿をしぼり込む（フィルター）

操作：画面下の「フィルター」ボタンをタップ → 画面の下からしぼり込みメニューが表示されます → 条件を選んでタップ
  • 自分の投稿 - 自分が書いた投稿だけ表示
  • そうだね多い - 「そうだね」がたくさんついた投稿
  • 返信多い - たくさん引用されている投稿
  • 削除済み - 削除された投稿
  • URL付き - リンクを含む投稿
  • 画像付き - 画像や動画がある投稿
  • キーワード - 入力したキーワードを含む投稿

スレッド内をキーワードで検索する

操作：画面右上の虫眼鏡アイコンをタップ → 上部に検索欄が表示される → キーワードを入力 → 見つかった件数が表示され、投稿内の該当箇所がハイライトされます。▲ ▼ ボタンで前後の結果に移動できます。✕ ボタンで検索を終了します

音声読み上げ

操作：画面下の「設定」→「読み上げ」をタップ → 再生ボタンで読み上げ開始。一時停止・停止・前後の投稿に移動する操作ができます

ギャラリー（画像・動画の一覧）

操作：画面下の「ギャラリー」ボタンをタップ → 画面下からスレッド内のすべての画像・動画がサムネイルで表示されます → 画像をタップで拡大表示 → 拡大画面では左右スワイプで前後に移動、2本指で拡大・縮小 → 保存ボタンで個別に端末へ保存

動画の再生

動画のサムネイルをタップすると再生が始まります。MP4、M4V、MOV 形式に対応しています。

WebM 形式の動画もサムネイル（小さなプレビュー画像）が表示されますが、端末や OS のバージョンによってはサムネイルが表示されないことがあります。その場合は代わりに動画アイコンが表示されます。

書き込み（返信・スレッド作成）

返信する

操作：スレッド画面の下にある「返信」ボタンをタップ → 画面下から書き込みフォームがスライドして表示されます

入力欄はすべて任意です。必要なものだけ入力してください。

  • 名前 - 表示される投稿者名
  • メール - メールアドレス（sage などにも使われます）
  • 題名 - 投稿の題名
  • 本文 - 書き込む内容
  • 画像・動画 - 1つまで添付できます
  • 削除キー - 投稿を後から削除するためのパスワード（最大8文字）
削除キーは前回使ったものが自動で入力されます。投稿を後で削除する可能性がある場合は、必ず入力しておきましょう。

特定の投稿を引用して返信する

操作：引用したい投稿の本文をタップ → 返信フォームが開き、その投稿番号が自動で引用欄に入ります

画像・動画を添付する

操作：返信フォーム内のクリップアイコン（添付ボタン）をタップ → 写真ライブラリまたは「ファイル」アプリが開く → 画像や動画を選ぶ

添付するファイルの選び方は3種類あり、設定で変更できます。

  • メディア - 写真ライブラリから選ぶ（初期設定）
  • ドキュメント - 「ファイル」アプリから選ぶ
  • 毎回確認 - 添付するたびにどちらを使うか選ぶ

新しいスレッドを立てる

操作：スレッド一覧画面の下にある「スレ立て」ボタンをタップ → ダイアログが表示される → タイトルと本文を入力して投稿

自分の投稿を削除する

操作：削除したい自分の投稿を長押し →「削除」を選ぶ → 削除キー（パスワード）を入力して確認
削除キーを忘れてしまった場合、投稿を削除することはできません。

スレッドや画像を保存する

スレッド全体を保存する

操作：スレッド画面の下にある「保存」ボタンをタップ → 保存が始まり、進み具合がダイアログで表示されます。途中で「キャンセル」ボタンを押すと中止できます

保存したスレッドは、インターネットに繋がっていなくても読めます。

画像・動画を1つずつ保存する

操作：画像をタップして拡大表示 → 画面上の保存ボタンをタップ。またはギャラリー表示中に保存ボタンをタップ

自動保存

設定で「バックグラウンド更新」を有効にすると、最近見たスレッドが自動的に保存されます。スレッドが消えてしまう前に取っておきたいときに便利です。

保存先を変える

操作：板管理画面の右上 ⋮ →「設定」→ 保存先の設定で変更
  • 手動入力 - 保存先のフォルダを文字で直接入力する
  • フォルダ選択 - フォルダ選択画面で保存先を選ぶ
フォルダ選択で選んだ保存先は、アプリを閉じても記憶されます。次回も同じフォルダに保存されます。

保存したスレッドを見る

操作：板管理画面の右上 ⋮ →「保存済み」をタップ → 保存済みスレッドの一覧が表示されます → スレッドをタップで閲覧

それぞれのスレッドには保存状態（完了・保存中・失敗・一部のみ）、投稿数、画像数、動画数、使用容量が表示されます。不要なスレッドは削除アイコンをタップして削除できます。

閲覧履歴

履歴を開く

操作：板管理画面・スレッド一覧画面では、画面左上の三本線アイコン（☰）をタップ。または画面の左端から右にスワイプすると、左から履歴パネルがスライドして表示されます

履歴パネルの構成

  • 上部 - 最近見たスレッドがサムネイル付きカードで一覧表示されます
  • 下部 - 「板」「更新」「一括削除」「設定」の4つのアイコンが並んでいます

履歴パネルの操作

  • スレッドカードをタップ → そのスレッドをもう一度開く
  • スレッドカードを右にスワイプ → その履歴を削除
  • 下部の「更新」アイコンをタップ → 履歴のスレッドに新着がないか確認
  • 下部の「一括削除」アイコンをタップ → すべての履歴をまとめて消す

履歴パネルを閉じる

操作：履歴パネルの右側（暗くなっている部分）をタップ。または画面の左方向にスワイプ

バックグラウンド更新（自動更新）

設定で有効にすると、アプリを開いていないときでも、履歴にあるスレッドの新着を自動でチェックします。更新のタイミングは iOS が自動的に管理します。

1回の自動更新で最大120件のスレッドを確認できます。
この機能を使うには、iPhone の「設定」→「一般」→「Appのバックグラウンド更新」で futacha が有効になっている必要があります。また「低電力モード」中は動作しないことがあります。

見たくない投稿を隠す（NG機能）

「NG」とは、指定した条件に当てはまる投稿を自動的に非表示にする機能です。

NGワード

特定の言葉を含む投稿を隠します。たとえば「ネタバレ」をNGワードに登録すると、「ネタバレ」を含む投稿が見えなくなります。

NGヘッダー

特定の名前やIDの投稿者を隠します。迷惑な投稿者がいるとき、その名前やIDを登録すると、その人の投稿が見えなくなります。

NGの登録方法

  • 投稿から直接 - 投稿を長押し →「NG登録」をタップ（いちばん簡単な方法）
  • スレッド画面のNG管理 - 画面下の「設定」→「NG管理」→ 下からスライドするシートでワードや名前を追加・削除
  • カタログのNG管理 - 画面右上 ⋮ →「NG管理」→ 下からスライドするシートでワードを追加・削除

設定

開き方：板管理画面の右上 ⋮ →「設定」をタップ

動作の設定

項目説明初期設定
バックグラウンド更新アプリを閉じている間も、履歴のスレッドを自動で更新するオフ
広告表示広告の表示・非表示を切り替えるオン
軽量モード画面の表示を簡略化して動作を軽くするオフ

ボタンの配置を変える

画面をスクロールすると「カタログメニュー」「スレッドメニュー」の項目があります。ここでボタンの表示・非表示や並び順を変えられます。

保存先の設定

  • 保存先の選び方 - 手動入力 / フォルダ選択 から選べます
  • 保存先のフォルダ - スレッドや画像を保存するフォルダを指定します

一時データの管理

  • 画像の一時データを削除 - 閲覧時にたまった画像の一時データを消して空き容量を増やす
  • 一時ファイルを削除 - その他の一時ファイルを削除する

容量の確認

自動保存されたスレッドの件数やサイズ、一時データの使用量を確認できます。

その他

  • Cookie管理 - サーバーとの通信に使われる情報の確認・削除
  • 作者 - 開発者の X（旧Twitter）で最新情報を確認
  • お問い合わせ - メールで意見や不具合を報告（admin@valoser.com）
  • 開発元 - ソースコードの公開ページ
  • プライバシーポリシー - 個人情報の取り扱いについて

よくある質問

板のアドレスがわかりません
Safari でふたば☆ちゃんねるの板を開き、アドレスバーの内容をコピーしてください。https://○○.2chan.net/○○/futaba.php のような形です。
スレッドを保存するには？
スレッド画面の下にある「保存」ボタンをタップしてください。全投稿と画像がまとめて保存されます。
保存したスレッドはどこから見る？
板管理画面の右上 ⋮ →「保存済み」で一覧が表示されます。ネットなしでも読めます。
迷惑な投稿者を非表示にしたい
その投稿を長押し →「NG登録」をタップしてください。それ以降、同じ名前やIDの投稿が自動で隠れます。
バックグラウンド更新が動いていない
(1) futacha の設定で「バックグラウンド更新」がオンか確認。(2) iPhone の「設定」→「一般」→「Appのバックグラウンド更新」で futacha が有効か確認。(3)「低電力モード」がオフか確認。なお、更新のタイミングは iOS が自動で決めるため、正確な間隔は保証されません。
画像が表示されない
ネット接続を確認してください。解決しない場合は、設定 →「画像の一時データを削除」を試してください。
自分の投稿を削除できない
書き込み時の削除キー（パスワード）が必要です。忘れた場合は削除できません。次回以降は自動入力されます。
WebM 動画のプレビュー画像が表示されない
WebM 形式の動画はプレビュー画像の自動生成に対応していますが、端末や OS のバージョンによっては生成できない場合があります。その場合、代わりに動画アイコンが表示されます。タップすれば再生できます。
保存先のフォルダを変えたい
設定 → 保存先の設定から変更できます。「フォルダ選択」で選んだフォルダは、アプリを閉じても記憶されます。
不具合の報告やお問い合わせ
設定 →「お問い合わせ」からメールを送れます。送信先: admin@valoser.com
